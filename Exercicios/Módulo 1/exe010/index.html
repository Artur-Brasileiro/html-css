Outras formatações!

Código-fonte:

Use a tag "code" para dar o mesmo espaço para cada caracter.
Exemplo: O comando document.getElementById('teste') é escrito em linguagem JavaScript
Sem a formatação: O comando document.getElementById('teste') é escrito em linguagem JavaScript

Também temos a tag "pre", que faz o code obedecer a indeintação do texto digitado.
Exemplo: 


num = int(input('Digite um número'))
if num % 2 == 0:
    print(f'O número {num} é par.')
else:
    print(f'O número {num} é ímpar.')

Citações:

Para citar algo, basta usar a tag q, automaticamente, o que estiver dentro da tag, irá receber aspas duplas, dando não só forma como sentido para a citação.
Exemplo: Como diria o pai de um amigo: o computador é um burro muito rápido!

Citações completas:

A tag blockquote é usada para citar frases, trechos, falas e etc, dando indeintação para os mesmos.
Exemplo: Segundo Thomas Jefferson, terceiro presidente dos Estados Unidos

A aplicação das leis é mais importante que a sua elaboração.

Abreviações:

Caso queira abreviar algo, é interessante você integrar o real significado na abreviação, para isso, basta ultilizar a tag abbr.
Exemplo: Estou estudando HTML e CSS, e estou adorando

Texto invertido:

Para inverter um texto, use a tag bdo, ela contém duas variações, da direita para a esquerda, e da esquerda para a direita
Exemplo: Eu estou aprendendo a criar coisas em HTML!
