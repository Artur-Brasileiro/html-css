Trabalhando com listas!

Listas ordenadas:

Tag ol para iniciar uma lista ordenada e li para adicionar elementos na lista.
Exemplo:

  1. Acordar
  2. Colocar os óculos
  3. Vestir roupa
  4. Preparar café
  5. Ligar computador

Listas não ordenadas:

Tag ul para iniciar uma lista não ordenada e li para adicionar elementos na lista.
Exemplo:

  • Acordar
  • Colocar óculos
  • Vestir roupa
  • Preparar café
  • Ligar computador

Listas de definição:

Ultilize a tag dl para iniciar uma lista de definição. Depois, basta usar a tag dt para específicar o termo e dd para especificar a descrição.
Exemplo:

Setup Gamer:
Se baseia no meu computador, um puro monstro capaz de rodar até a mãe do Ademir. ;)
Computador:
RX6600, Xeon 2666 v3, 32GB de RAM, 480GB de ROM.
