Usando links externos:

Para transformar algum termo em uma âncora, basta envolopa-lo com a tag a. Quando você clicar no link, automaticamente o seu site fecha e o link clicado abre, portanto, é viável usar target="_blank" para abrir uma nova guia ao clicar, mantendo seu site aberto. rel="external" serve para o site entender que o link é externo.
